# class-list

[![Greenkeeper badge](https://badges.greenkeeper.io/ForbesLindesay/class-list.svg)](https://greenkeeper.io/)

A cross browser class list

## Example

```
var elem = someElement
    , ClassList = require("class-list")
    , cl = ClassList(elem)

cl.add("foo")
cl.remove("bar")
cl.toggle("baz")
```

## Installation

`npm install class-list`
